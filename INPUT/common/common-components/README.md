# Common Components

Reusable UI component patterns shared across screens.

## Dialog Component
- Confirmation dialog (OK / Cancel)
- Error dialog (message + dismiss)
- Form dialog (input fields + submit)

## Loading States
- Skeleton loader for lists
- Spinner for actions
- Progress bar for multi-step processes

## Form Components
- Text input with validation
- Select / Dropdown
- Date picker
- File upload area

## Navigation
- Breadcrumb
- Tab navigation
- Sidebar menu
- Back button
